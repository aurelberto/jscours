En JS, obn déclare une variable avec les mots-clés let, const et anciennement var( toujours utilisé dans du vieux code parcontre ne plus l'utiliser) voir exemple en base.js

Les boucles

Une boucle est une structure qui permet de répéter un bloc de code tant qu'une condition est vraie.

C'est une façon automatique de répéter des instructions sans devoir les écrire plusieurs fois

Les Types de boucles

  • Boucle While
  • Boucle Do...while
  • Boucle For
  • Boucle For...of
  • Boucle For...in

La boucle while

On l'utilise quand on ne connait pas forcément à l'avance le nombre de tours

            
                while (condition){
                    
                }

La boucle Do...While

C'est comme la boucle while, mais elle s'éxécute au moins une fois avant de tester la condition

            
                let passeword;
                do (condition){
                   //
                } while (condition);

La boucle For

La plus classique et la plus utilisé quand on connait à l'avance le nombre de répétition

            
                for (initialistation; condition; incrémentation) {
                    // code à éxécuter
                }

1. Introduction à Javascript

C'est quoi le JavaScript?

Le Javascript est un language de programation qui donne de l'interactivité aux pages web

  • HTML = structure ou squelette
  • CSS = styles ( les couleurs, la mise en page, ...)
  • Le JS = La logique ( les récations à un utilistaues, des calculs et la dynamique de la page)

Où s'exécute le jS?

  • Dans le navigateur(chrome, firefox, safari ect...)
  • Sur un serveur qui tourne avce node.JS -> il ne sera plus un language front-end mais back-end comme le PHP ou le python ect...

A retenir : même langage, mais deux environnemets différents

Premier pas : console balise script

  1. Ouvrir votre navigateur préféré
  2. Clique droit -> inspecter, ou si vous êtes sur windows F12
  3. Aller dans l'onglet console
